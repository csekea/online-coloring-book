## About The Project

The Online coloring book allows you to fill line drawings with color. You can create, save, modify and delete these colored pictures. This full-stack application uses React on the frontend, ExpressJS on the backend and MongoDB as a database.

## Getting Started

Please see the core dependencies and installation steps below:

### Prerequisites

* [Node.js](https://nodejs.org/en/download)
* [MongoDB](https://www.mongodb.com/docs/manual/administration/install-community/)
* Integrated development environment(IDE) - e.g. [Visual Studio Code](https://code.visualstudio.com/Download)
* [MongoDB Compass](https://www.mongodb.com/try/download/compass)

### Installation

1. Clone the repository `git clone https://github.com/csekea/online-coloring-book.git`
3. Install an integrated development environment(IDE)
4. Install MongoDB
5. Install MongoDB Compass for database visualization
6. Open the project in Visual Studio Code: `File / Open folder... --> online-coloring-book`
8. Open a terminal in Visual Studio Code: `Terminal / New Terminal`
9. Install npm packages by entering `npm install` into the terminal
10. Navigate to the `server` folder by entering `cd server`
11. Start the Node.js server by entering `npm start`
12. Open another terminal by clicking to the 'plus' sign in the top right corner of the terminal window
13. Navigate to the `client` folder by entering `cd client` into this terminal 
14. Start the React.js server by entering `npm start`
15. This will start the project which you can reach in your browser, on the following URL: `http://localhost:3000/`
